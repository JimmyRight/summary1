SGI IRIX 固有のサービス
***********************

この章で記述されているモジュールは、SGI の IRIX オペレーティングシステ
ム (バージョン 4と5) 固有の機能へのインターフェイスを提供します。

* "al" — SGIのオーディオ機能

  * コンフィギュレーションオブジェクト

  * ポートオブジェクト

* "AL" — "al" モジュールで使われる定数

* "cd" — SGI システムの CD-ROM へのアクセス

  * Player オブジェクト

  * Parser オブジェクト

* "fl" — グラフィカルユーザーインターフェースのための FORMS ライブラ
  リ

  * "fl" モジュールに定義されている関数

  * Form オブジェクト

  * FORMS オブジェクト

* "FL" — "fl" モジュールで使用される定数

* "flp" — 保存された FORMS デザインをロードする関数

* "fm" — *Font Manager* インターフェース

* "gl" — *Graphics Library* インターフェース

* "DEVICE" — "gl" モジュールで使われる定数

* "GL" — "gl" モジュールで使われる定数

* "imgfile" — SGI imglib ファイルのサポート

* "jpeg" — JPEG ファイルの読み書きを行う
