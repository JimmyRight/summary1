Débogger et profiler
********************

Ces bibliothèques sont là pour vous aider lors du développement en
Python : Le débugger vous permet d’avancer pas à pas dans le code,
d’analyser la pile d’appel, de placer des points d’arrêts, … Les
profilers exécutent du code et vous donnent un rapport détaillé du
temps d’exécution, vous permettant d’identifier les goulots
d’étranglement dans vos programmes.
